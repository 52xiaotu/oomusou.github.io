如何安裝其他 .NET Core Project Template ?

發現更多 C# 與 F# 的有趣應用
Contents
  1. 列出目前所有 Project Template
  2. 安裝 Project Template
  3. Conclusion
  4. Reference

.NET Core SDK 已經內建提供不少 project template，但事實上 .NET Core 官網還提供不少 project template，可自行另外安裝。

列出目前所有 Project Template

  • -l : list ，使用 dotnet new 加上 -l 參數，將列出目前系統所安裝的所有 project template。

事實上在 .NET Core 官網的 Available templates for dotnet new，還準備了不少其他 project template 可供下載。

安裝 Project Template

以 F# 為例，.NET Core 官網也提供不少有趣的 project template，如 Fable，可以讓我們寫 F#，但最後編譯成 JavaScript。

我們以 Fable 為例安裝新的 project template。

  • -i : install，使用 dotnet new -i 安裝新 project template

  1. 輸入 dotnet new -i "Fable.Template::*" 安裝 Fable project template

  1. 安裝完後會自動執行 dotnet new -l ，顯示 Fable project 已經安裝成功

Conclusion

  • 若想知道 C# 與 F# 還有什麼應用，可上 Available templates for dotnet new 發現更多有趣的 project template

Reference

.NET Foundation, Available templates for dotnet new
